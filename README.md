# Zuma (Python / pygame)

Классическая игра **Zuma**, реализованная на Python от PAI и littledarkness.

## Возможности
- Спиралевидная траектория шаров
- Вращающаяся лягушка
- Уничтожение одноцветных цепочек (3+)
- Череп (конец пути)
- Счёт
- Бонусы (архитектура готова)
- Несколько уровней (через конфиги)
- Юнит-тесты

## Установка

pip install -r requirements.txt
